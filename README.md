# cdm-sdk

How to build the wheel file locally

1. Move to your package directory
 cd C:\Ashutosh\Boston\cdm-sdk

2. Create a python envronment .venv 
    python -m venv .venv
    .\.venv\Scripts\Activate.ps1


4. Insall the latest version of pip
    python -m pip install --upgrade pip


5. Installing the lastest version of build
    python -m pip install build


6. Creating the python artifacts
    python -m build
